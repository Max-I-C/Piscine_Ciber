# 42 Cybersecurity Piscine

In this repository, you will find all the projects completed during the 42 Cybersecurity Piscine (Outer Core).

The piscine is divided into multiple projects that must be completed sequentially, each focusing on a specific aspect of cybersecurity.

# Projects
## Arachnida
* This project consists of building your first web scraper (bot).
* It also includes developing a program to analyze images, extracting metadata and uncovering hidden information.

## ft_otp
* In this project, you learn how Two-Factor Authentication (2FA) works.
* The goal is to generate time-based one-time passwords (TOTP) using an encrypted key, producing a code that changes every few seconds.

## ft_onion
* The goal of this project is to deploy a website on the Tor network, making it accessible via an .onion address.

## Reverse_me
* This project introduces reverse engineering.
* You will analyze a compiled program and attempt to reconstruct its potential source code.

## Stockholm
* This project involves creating a ransomware-like program.
* It focuses on recursively encrypting files to understand how such attacks work.

## Inquisitor
* This project consists of performing a Man-in-the-Middle (MITM) attack using techniques such as MAC address spoofing and traffic redirection.

## Vaccine
* The final project of the cybersecurity piscine.
* You will build a tool that analyzes a website (via a given URL) and detects potential SQL injection vulnerabilities.

## Notes
All projects are part of the 42 learning methodology.
They are designed for educational purposes only, to better understand cybersecurity concepts and threats.
